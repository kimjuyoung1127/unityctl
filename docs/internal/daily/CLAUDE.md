# docs/daily/

일일 실행 로그.

주의:
- 이 폴더의 문서는 당시 시점 기록이다.
- 현재 코드/문서 상태 판단에는 사용하지 않는다.
- 현재 기준은 `docs/DEVELOPMENT.md`, `docs/status/PROJECT-STATUS.md`, `docs/status/PHASE-EXECUTION-BOARD.md`를 본다.

## 파일명 컨벤션
- 폴더: `MM-DD/`
- 파일: `module-{모듈슬러그}.md`
- 예시: `03-17/module-ipc-transport.md`

## 규칙
- 작업 완료 시 해당 날짜 폴더에 로그 작성.
- 주간 롤업(`docs/weekly/`)으로 집계.
